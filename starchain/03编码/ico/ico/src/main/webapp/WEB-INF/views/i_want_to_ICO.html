如何在 ICO 上发起项目？

ICOINFO 欢迎经验丰富、具有潜力的区块链项目一同参与，透过 ICOINFO 的平台推向全世界，如果你是符合条件的团队，请先参考以下申请条件，并提交申请表单，我们会有专人与你联系。

申请条件

一、请先确定你的项目符合 ICO 原则。

二、请先确定你的项目 ICO 预计开始的时间、发布策略，及 ICO 的目标金额及货币种类。

三、请先确定你的项目 ICO 成功后，后续的规划及代币发放的计划。

四、请先确定你的项目有详尽的商业计划书、白皮书、团队介绍等，各项能提高项目发展潜力的证明。

申请流程

填写表单 → 专业团队审核（专人联系）→ 审核通过 → 准备项目上线

*资料越详尽，申请通过的可能性越高，请你准备充足后再进行申请。
